What is the advantage of using a text editor 

There are many advatages of using a text editor. Some of the main advantages are speed and eliminating errors. Using a text editor is quite fast and helps a new user to learn. The text editor also helps with seeing the final product so you can eliminate errors such as bad grammar, and mispellings. 


Is there an advantage of using a relative vs an absolute URL

Both relative and absolute URL have advantages and disadvantages. The main advantage of a relative URL is that it is shorter than an absolute URL thus, easier to use when you are developing a new website. Other advantages includes potentially faster load times depending on the complexitity of the website.
